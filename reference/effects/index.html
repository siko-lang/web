Effect Handlers

Effect handlers allow you to define and handle computational effects in a type-safe way.

Effect Definition

effect Reader[T] {
  fn ask() -> T
}

effect Writer[T] {
  fn tell(value: T)
}

Using Effects

fn use_reader() {
  let value = ask();
  println("Value: ${value}");
}

Handling Effects

Effects are handled using with blocks:

fn provide_value() {
  with Reader.ask = || { 42 } {
    use_reader();  // ask() will return 42
  }
}

Multiple Effects

Handle multiple effects at once:

fn example() {
  with Reader.ask = || { 10 },
       Writer.tell = |x| { println("Told: ${x}") } {
    let value = ask();
    tell(value * 2);
  }
}

Effect Polymorphism

Functions can be polymorphic over effects:

fn generic_computation[E: Effect](/* ... */) {
  // Can use effect E
}
